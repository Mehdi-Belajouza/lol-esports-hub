# Rank Icons Setup

This directory contains the rank icons for all supported games. Please download the rank icons from the official sources and place them in the appropriate folders.

## Directory Structure

```
public/assets/ranks/
├── lol/           # League of Legends rank icons
├── valorant/      # Valorant rank icons
└── csgo/          # CS:GO/CS2 rank icons
```

## Image Specifications

- Format: PNG with transparency
- Recommended size: 256x256px or larger
- Background: Transparent
- File naming: lowercase with hyphens (as shown above)
